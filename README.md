# dotted-sign

## Features:
- Vue 3
- Vite
- PNPM
- Tailwind CSS
- Eslint
- Pinia / Pinia-plugin-persistedstate
- Fabric.js
- jspdf
- pdfjs-dist

## Getting Started
Install and start dev server (Node.js version v16.0.0)

```sh
pnpm install
pnpm dev
```
